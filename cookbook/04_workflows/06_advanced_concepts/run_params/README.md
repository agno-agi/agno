# 04_workflows/06_advanced_concepts/run_params

## Scope
Runnable workflow examples under: cookbook/04_workflows/06_advanced_concepts/run_params

## Files
- workflow_metadata.py: Demonstrates passing metadata from workflow to agents.
- workflow_dependencies.py: Demonstrates dependency injection through workflows.
- workflow_context_flags.py: Demonstrates context control flags at the workflow level.
- workflow_all_params.py: Demonstrates all run-level params together in a realistic pipeline.

## Prerequisites
- Activate the demo environment: .venvs/demo/bin/python
- Load API keys with direnv allow (requires a local .envrc file).
