# Databricks

Cookbook examples for `cookbook/90_models/databricks`.

## Prerequisites

Export the Databricks model credentials:

```bash
export DATABRICKS_HOST=***
export DATABRICKS_TOKEN=***
export DATABRICKS_CHAT_ENDPOINT=***
```

## Files

- `basic.py` - Sync and streaming chat model example.

## Run

```bash
.venvs/demo/bin/python cookbook/90_models/databricks/basic.py
```
